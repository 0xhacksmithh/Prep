# 2. Is Node.js single-threaded?

Yes, Node.js is single-threaded, but it handles multiple requests using asynchronous, non-blocking I/O.

**Important:**

- JS runs on one thread

- Heavy tasks are offloaded to libuv thread pool

```
setTimeout(() => {
  console.log("Async task");
}, 0);

console.log("Main thread");
```
